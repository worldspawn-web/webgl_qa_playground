## Airtest Naming

```ts
// Folder Structure:
airtest/<GameName>/<Platform>/<Sub-Platform>/<User>/

// Folder & File Naming:
<Name>-<Android/iOS>-<Browser>-<Resolution>.air/...
```

Example:

```
airtest/Hell Merge/Web/Yandex/worldspawn/android-googlechrome-1080x2400.air/...
```
