## Airtest Naming

```ts
// Folder Structure:
airtest/<GameName>/<Platform>/<Sub-Platform>/<User>/

// Folder & File Naming:
<Name>-<Android/iOS>-<Browser>.air/...
```

Example:

```
airtest/Hell Merge/Web/Yandex/worldspawn/Authorized-Android-Chrome.air/...
```
